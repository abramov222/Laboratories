Контрольные вопросы:
Что такое сокет и чем TCP отличается от UDP?
Сокет — программная конечная точка для сетевого взаимодействия. TCP — надежный, с установкой соединения, гарантирует доставку и порядок данных. UDP — ненадежный, без соединения, быстрее, но без гарантий доставки и порядка.

Как сервер может одновременно обслуживать несколько клиентов?
Через многопоточность: для каждого нового клиента создается отдельный поток (или используется пул потоков), который обрабатывает его запросы независимо.

Для чего используется SLF4J?
SLF4J — фасад (интерфейс) для логирования, который позволяет абстрагироваться от конкретной реализации (Logback, Log4j) и легко менять ее без изменения кода приложения.

Как работает конфигурация Logback?
Через XML-файл logback.xml, где задаются аппендеры (куда писать логи), форматы вывода, уровни логирования (INFO, ERROR и т.д.) и правила маршрутизации сообщений (например, ошибки — в отдельный файл).

Зачем разделять сообщения на личные и широковещательные?
Для приватности и эффективности: личные сообщения — конфиденциальный обмен между двумя пользователями; широковещательные — для общих объявлений или группового обсуждения, экономят трафик и упрощают коммуникацию.
